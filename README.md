# AI YouTube Video Analyzer

An AI-powered application that analyzes YouTube videos and generates structured insights such as:

* Video Overview
* Content Structure
* Timestamp Suggestions
* Key Takeaways
* Summary
* Content Organization

## Features

* Analyze YouTube videos using AI
* Generate detailed summaries
* Create timestamp suggestions
* Extract important insights
* Streamlit-based web interface

## Tech Stack

* Python
* Streamlit
* Agno
* Groq
* YouTube Transcript API

## Installation

```bash
pip install -r requirements.txt
```

## Run Locally

```bash
streamlit run youtube_analyzer.py
```
